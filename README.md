# Drawing Board

A simple web drawing board made with Vanilla JS.  
[[Demo]](https://coachoox.github.io/js-drawing-canvas/)  

## Preview

![](./images/preview.png)

## Features

-   Drawing something (Line)
-   Fill canvas (Paint tool)
-   Save image to local
-   Change brush size
-   Change color
